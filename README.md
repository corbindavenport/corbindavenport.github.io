This is the website for [Corbin Davenport](https://corbin.io), made with [Eleventy](https://www.11ty.dev/) and [Simple.css](https://simplecss.org/). It's built with GitHub Actions and deployed on GitHub Pages.

## Local development

How to run locally:

```
npm install
npm start
```

The site will be accessible at [localhost:8081](http://localhost:8081) in your web browser.
